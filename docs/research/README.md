## 使用方法

1. **第一遍通读（约 90 分钟）**：按 00 → 06 顺序快速浏览，建立全景。
2. **第二遍按月执行**：跟随[第三部分 12 个月路线图](./03-12月路线图.md)，每月对照一次。
3. **每月修订**：把新发现的学者、新意识到的问题加进对应文档。文档本身是一个**持续重构的 PKM**。
4. **链接驱动阅读**：每篇文档底部的"延伸阅读"指向相关深度研究。沿着链接走，会比线性阅读得到更多。
