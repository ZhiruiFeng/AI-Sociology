# 第五部分：研究方法与工具

## 5.1 跨学科研究方法

### (A) 概念图谱法

每读完一本书或一组论文，画一张**概念关系图**——节点是概念，连线是它们的关系（推导、对立、子集、案例）。

半年后回看你的概念图集合，会发现真正的"自己的思路"自然涌现。

**操作要点**：
- 节点 = 概念，连线 = 关系（推导、对立、子集、案例）
- 用颜色标注学者来源
- 每月做一次"图合并"——把不同学派的概念图叠加，看交集与冲突

### (B) 对照矩阵法

把一个 AI 现象放在不同学科的视角下分析。

**示例：ChatGPT 上线**

| 维度 | 社会学问 | 伦理学问 | 未来学问 | 技术哲学问 |
| --- | --- | --- | --- | --- |
| **问题** | 谁因此受益/受损？ | 是否符合知情同意？ | 这是哪条轨迹的早期信号？ | 人/机界面发生什么改变？ |
| **核心概念** | 劳动重组、数据殖民 | 自主性、误导风险 | S 曲线、转折点 | 外化认知、媒介物质性 |
| **证据类型** | 访谈、田野、数据 | 案例分析、原则演绎 | 外推、专家预测 | 现象学描述、文本分析 |

**进阶用法**：把同一现象放在**历史对照 + 学科视角**的二维矩阵里（见 [第二部分 · 跨章节小结](./02-历史镜鉴.md#跨章节小结四类对照如何配合使用)）。

### (C) 历史对照法

已在 [第二部分](./02-历史镜鉴.md) 详述。

**要点**：**找差异而不是找相似**。

```
错误：AI ≈ 电力，所以会复制电力的故事
正确：AI 和电力的关键差异是 X，所以 X 会改写脚本
```

### (D) 反事实分析法

问"如果...会怎样？"

**示例**：
- "如果 LLM 一开始就是开源的，AI 产业格局会怎样？"
- "如果 RLHF 没有被发明，对齐问题会以什么形式出现？"
- "如果中美在 2020 年签了 AI 协议，今天的地缘格局会怎样？"

通过这种思想实验，能识别"**必然的**"和"**偶然的**"。技术从业者最熟悉的方法。

## 5.2 信息源管理

### RSS / 邮件订阅

| 类别 | 推荐源 |
| --- | --- |
| **学术** | Google Scholar Alerts（5–10 个关键词） |
| **智库** | Brookings、Carnegie、CSET 的 AI 相关邮件 |
| **Substack (英文)** | Asterisk Magazine（EA/AI 安全）、Astral Codex Ten（哲学）、Henry Farrell（Crooked Timber）、Don't Worry About the Vase（理性主义）、Marcus on AI（怀疑论） |
| **中文** | 远川研究所、混沌大学、机器之心、吴军（公众号/Newsletter） |

### Twitter/X 列表

建立 **5 个分类列表**：

1. **批判算法学者**：Crawford, Whittaker, Benjamin, Eubanks, Noble...
2. **AI 安全**：Russell, Christiano, Karnofsky, Cotra, MacAskill...
3. **技术哲学**：Floridi, Vallor, Coeckelbergh, Hui...
4. **经济学家**：Acemoglu, Brynjolfsson, Mollick, Goldin...
5. **前沿工程师**：用来**对照**他们的世界观和人文学者的差异

### 笔记系统

强烈建议建立一个 **Zettelkasten 风格的卡片笔记系统**（Obsidian / Logseq / Roam）。

**三条规则**：

1. **每张卡只一个想法**（atomic notes）
2. **用"自己的话"重述**，而不是摘抄
3. **卡与卡之间建立链接**——你后来会写出来的所有原创内容，都是从链接里涌现的

## 5.3 写作与发表策略

### 写作的三个层级

| 层级 | 形式 | 频率 |
| --- | --- | --- |
| **L1 个人笔记** | 卡片、读书笔记 | 每天 |
| **L2 公开短文** | 博客 2000–5000 字 | 每月 1–2 篇 |
| **L3 深度长文** | 论文规格 10000+ 字 | 每季度 1 篇 |

### 发表阵地（按门槛递增）

```
个人博客 / Substack / 微信公众号  → 零门槛、训练表达
       ↓
Medium / The Gradient / Asterisk Magazine  → 邀请制或低门槛投稿
       ↓
预印本 (SSRN, arXiv cs.CY)  → 免审稿，可被引用
       ↓
期刊 (AI & Society, Big Data & Society)  → 审稿 6–12 月，建立学术身份
       ↓
出版社 (MIT Press, Yale UP, Princeton UP)  → 书，5–10 年视野
```

## 5.4 工具栈速查

### 文献管理

- **Zotero**（开源、推荐）
- **Connected Papers**（论文关系图，免费）
- **Research Rabbit**（论文发现 + 追踪）

### 笔记与写作

- **Obsidian / Logseq**（双向链接笔记）
- **Notion**（团队协作）
- **Scrivener**（长文写作）

### AI 辅助研究

- **Claude / ChatGPT** —— 概念辨析、文献综合
- **Elicit** —— 学术论文搜索
- **NotebookLM** —— 多文档对话与笔记
- **Perplexity** —— 实时检索

### 公开发表

- **Substack**（英文/中文均可）
- **微信公众号**（中文）
- **知乎专栏**（中文长文）
- **SSRN / arXiv cs.CY**（学术预印本）
