# 附录 B：实用工具与平台

## 文献管理

- **Zotero**（开源、推荐）
- **Connected Papers**（论文关系图，免费）
- **Research Rabbit**（论文发现 + 追踪）

## 笔记与写作

- **Obsidian / Logseq**（双向链接笔记）
- **Notion**（团队协作）
- **Scrivener**（长文写作）

## AI 辅助研究

| 工具 | 用途 |
| --- | --- |
| Claude / ChatGPT | 概念辨析、文献综合 |
| Elicit | 学术论文搜索 |
| NotebookLM | 多文档对话与笔记 |
| Perplexity | 实时检索 |

## 公开发表

- **Substack**（英文/中文均可）
- **微信公众号**（中文）
- **知乎专栏**（中文长文）
- **SSRN / arXiv cs.CY**（学术预印本）
