│       ├── README.md                # 总索引
│       ├── 00-序言-研究定位.md       # 序言：研究定位
│       ├── 01-四学科视角理论地图.md   # 社会学 / 伦理学 / 未来学 / 技术哲学
│       ├── 02-历史镜鉴.md            # 通用技术 / 媒介 / 政经 / 思想范式
│       ├── 03-12月路线图.md          # 按月可执行的 12 个月计划
│       ├── 04-分层级必读清单.md       # 入门 12 本 + 进阶 15 本 + 经典 12 本
│       ├── 05-研究方法与工具.md       # 跨学科方法 + 工具栈

## 快速开始

**只看文档**：直接进入 [`docs/research/`](./docs/research/) 阅读。

**启动网站**：

```bash
cd web-app
npm install
npm run dev
# 打开 http://localhost:3000
```
